# Screenshots

This directory contains screenshots for the project README.

## Main Screenshot

**main-interface.png** - The primary screenshot showing the full application interface.

### How to Capture

1. Run the application: `pnpm run dev`
2. Open http://localhost:3088
3. Scan a directory and start an agent task
4. Take a screenshot showing all three panels
5. Save as `main-interface.png` in this directory

### Recommended Content

Show:
- Repository list with some discovered projects
- An active agent running in the center panel
- Task tabs in the right sidebar
- Real agent output (thinking, tool calls, responses)

### Tips

- Use a high-resolution display for crisp screenshots
- Ensure the UI is fully loaded before capturing
- Show meaningful content with real repository names
- Capture at ~1200-1400px width for best visibility
- PNG format for quality
